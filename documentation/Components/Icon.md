# Icon
JellyBean extends FluxUI's [icon component](https://fluxui.dev/components/icon), by using [FontAwesome's Pro icons](https://fontawesome.com/v6/search?ip=classic&f=classic&s=light&o=r).

## Usage
Simply use the icon name from Font Awesome instead of the heroicons one.

```html
// <flux:icon.{icon} />
<flux:icon.user />
```

## Notes
1. [Variants](https://fluxui.dev/components/icon#variants) are not supported.
